# NBA Arena Performance & Fan Experience Optimization

## Overview

This project analyzes NBA arena attendance, team performance, and fan engagement trends using Python and interactive dashboards. The goal is to identify key factors that drive fan satisfaction, attendance rates, and revenue performance in sports venues.

The dashboard provides insights into operational metrics and fan behavior using data visualization and analytics techniques commonly used in Business Intelligence and Data Analytics roles.

---

## Objectives

* Analyze attendance and team performance trends
* Measure fan engagement and satisfaction
* Identify drivers of arena revenue and operational performance
* Demonstrate real-world data analytics and dashboard development skills

## Dataset

The dataset simulates NBA game and arena performance data, including:

* Game dates
* Teams and opponents
* Attendance numbers
* Arena capacity
* Team scores
* Fan satisfaction ratings
* Merchandise and concession sales
* Revenue metrics

This dataset structure reflects real-world sports analytics and venue performance reporting.

## Key Metrics Calculated

* Revenue per Fan

* Fan Satisfaction Score
* Engagement Index

## How to Run the Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run app.py
```
